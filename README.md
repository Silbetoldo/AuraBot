# Landing page do AuraBot

Abra `index.html` no navegador para visualizar a página.

O checkout utilizado em todos os botões é:

`https://pay.hotmart.com/H106657467L?checkoutMode=2`

Antes de publicar, confirme no cadastro da oferta da Hotmart que o preço exibido no checkout é `R$ 19,90`.
